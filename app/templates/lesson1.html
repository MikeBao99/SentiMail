Let’s say you are looking through a phonebook for a particular person’s name. You would just go through the book one name at a time until you found that person’s name. This process is called the “linear search”. Linear searches work on any list of variables.

Linear search always works, but it is very slow. If we can somehow sort the list of numbers, we can find a faster searching algorithm. There are many ways to sort a list of numbers.
