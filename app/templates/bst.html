One useful implementation of the tree is in binary search trees (BST). Binary search trees are trees that have been organized so that binary search is easy to run. Each node in a binary search tree keeps track of two child nodes (left and right) and its own value. The left child is always smaller than the parent node, while the right child is always larger than the parent node. This way, for each node in the tree, all nodes underneath to the left are smaller, while all nodes underneath to the right are larger.

We can construct the tree by creating nodes and assigning them as the left and right child nodes of existing nodes, making sure that the left node is smaller and the right node is larger than the parent node.
