Now that we have constructed a binary search tree, we can perform binary search on our tree. We can implement our binary search using recursion. Each recursive step checks whether the current node contains the value we are searching for. If it does, then we are done. Otherwise, it moves to the left node if the value is less than the node’s value and to the right node if the value is greater than the node’s value.
